#  Student Management System

A complete, feature-rich Student Management System built with Flask, SQLite, and modern web technologies.

##  Features

###  Dashboard
- View all students in a beautiful card layout
- Real-time statistics and metrics
- Quick actions for each student (View, Edit, Delete)

###  Student Management
- **Add Students**: Comprehensive form with validation
- **Edit Students**: Update student information
- **View Details**: Complete student profile view
- **Delete Students**: Safe deletion with confirmation

###  User Interface
- Modern, responsive design
- Light/Dark theme toggle (persistent)
- Smooth animations and transitions
- Mobile-friendly layout

###  Technical Features
- Full CRUD operations
- Form validation (client & server side)
- Flash notifications
- Keyboard shortcuts
- Auto-dismissing messages
- SQLite database with sample data
